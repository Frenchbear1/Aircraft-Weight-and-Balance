# Aircraft Weight and Balance

A lightweight, single‑page weight and balance calculator for aircraft. It saves aircraft profiles locally and plots takeoff/landing CG against a configurable envelope.

## Features
- Local aircraft profiles with autosave, duplicate, and delete
- Editable stations with moments and fuel burn (taxi + trip in gallons)
- Takeoff/landing CG and weight calculations
- Custom CG envelope editor with draggable points
- Visual graph with grid/labels and out‑of‑envelope warning
- Export/Import backup of all local data
